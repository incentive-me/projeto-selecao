# Back-end-Incentive

## como começar:

```bash
npm install
```

## DB:

```bash
docker compose up -d
```

## para iniciar as migrations

```bash
npx prisma migrate dev
```

## para iniciar localmente:

```bash
npm run start:dev
```

# Front-end-Incentive

## como começar

```bash
npm install
```

## para iniciar localmente:

```bash
npm run dev
```
