## Getting Started

Run developer mode

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

To build

```bash
yarn build
# and
yarn start : run server
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
